# 📚 나만의 도서관 (My Personal Library)

나만의 도서 대출 현황을 관리하고, 도서관 이용증과 좋아하는 작가를 한곳에서 관리할 수 있는 프리미엄 개인 도서관 관리 웹 앱입니다.

![Preview](mobile_qr.png)

## ✨ 주요 기능

- **📖 빌린 책 관리**: 도서 대출 및 반납 기한 관리 (D-day 자동 계산 및 연체 알림)
- **📷 바코드 스캔**: 카메라를 통한 ISBN 바코드 인식으로 도서 자동 등록 (Open Library API 연동)
- **🪪 도서관 이용증**: 모바일 도서관 이용증 사진 등록 및 회원번호 관리
- **🔔 스마트 알림**: 반납 예정일(D-3) 및 반납 당일 브라우저 알림 발송
- **✍️ 좋아하는 작가**: 관심 작가의 신간 검색 및 인터뷰/YouTube 영상 퀵 링크
- **🏛️ 추천 도서관**: 전국 주요 도서관 정보 필터링, **길찾기** 및 **주변 맛집(블로그)** 추천
- **📱 모바일 연동**: QR 코드로 접속하여 핸드폰에서 실시간 테스트 지원

## 🛠 사용된 기술

- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism UI), JavaScript (ES6+)
- **Storage**: LocalStorage (별도의 서버 없이 브라우저에 데이터 안전 저장)
- **Library**: `html5-qrcode` (바코드 스캔), `qrcodejs` (접속 QR 생성)
- **API**: Open Library API (도서 정보), Google Maps/Naver Maps (길찾기)

## 🚀 시작하기

1. 본 저장소를 클론(Clone)하거나 다운로드합니다.
2. `index.html` 파일을 크롬(Chrome) 등의 최신 브라우저로 엽니다.
3. **[중요]** 바코드 스캔 및 알림 기능을 사용하려면 `localhost` 환경이나 `HTTPS` 보안 연결이 필요합니다.

## 📱 모바일 테스트

앱 실행 후 헤더의 **'📱 모바일'** 버튼을 클릭하여 생성되는 QR 코드를 핸드폰으로 스캔하세요. (PC와 핸드폰이 같은 Wi-Fi에 연결되어 있어야 합니다.)
